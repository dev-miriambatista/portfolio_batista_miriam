  • Book Flix

    O Book Flix é um repositório online de livros desenvolvidos em PHP com o framework Senac\Mvc. Permite listar, filtrar e organizar livros por gênero, oferecendo uma interface estilizada e navegável. O sistema conta com rotas personalizadas e estrutura de login. As imagens dos livros são armazenadas por caminho, e os ícones do Bootstrap são integrados. O design responsivo é aprimorado com Flexbox e CSS, garantindo uma experiência otimizada em diversos dispositivos.

  • Decodificador

    O código JavaScript implementa um sistema de criptografia e descrição de texto, utilizando dois arrays: um com letras do alfabeto e outro com caracteres especiais correspondentes. As funções criptografarTexto e descriptografarTexto realizam a conversão entre os arrays. O sistema inclui botões para criptografar, descrever, copiar para a área de transferência e limpar campos. Os eventos são gerenciados com addEventListener para garantir uma interface interativa e responsiva ao usuário.
